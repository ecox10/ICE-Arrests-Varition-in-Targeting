# ICE-Arrests-Varition-in-Targeting

This replication package produces the main result from the paper "ICE Arrests, 2015-2026: Variation in Targeting, Method, and Geography." 

To replicate these results, users will need to download the data files saved in the "data" folder, set the paths in 00_setup.do (and the working drives in the R files) to match their directories, and run the files in the "analysis" folder in sequential order. More details on what each file contains are provided below.
